# PDF Splitter With Naming

A Python utility for splitting PDFs and naming the output files.

## Purpose

Splitting PDFs is more useful when the output files are named consistently. This project explores a document workflow utility for breaking PDFs apart while controlling the resulting file names.

## What This Shows

- PDF splitting workflow
- Output naming logic
- Python automation utility
